# Generador de Paletas

Aplicación web desarrollada con HTML, CSS y JavaScript que permite generar paletas de colores aleatorias en formato HEX y HSL.

---

## Características

- Generación automática de colores
- Conversión a formato HEX
- Conversión a formato HSL
- Interfaz interactiva


---

## Tecnologías utilizadas

- HTML5
- CSS
- JavaScript

```bash
📁 proyecto

### Descripción

- 📂 `assets/img` → Contiene los archivos que se utilizan en el proyecto como: imagenes y gifs.
- 📂 `css` → Contiene los estilos de la aplicación.
- 📂 `js` → Contiene la lógica y funcionalidades en JavaScript.
- 📄 `index.html` → Estructura principal de la página web.
- 📄 `generador-paleta.html` → Sección de generación de paletas
- 📄 `README.md` → Documentación del proyecto.
